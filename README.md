# Age of Empires Units

## For live demo: [Age of Empires Units on Firebase](https://age-of-empires-units.web.app/)

## Overview

Users are able to:

- Check all units
- Filter units by ages
- Filter units by costs (Food, Wood, Gold)
- Redirect to details page of each unit
